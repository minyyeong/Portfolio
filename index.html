PROJECT 01

KYOBO MUNGO

Book Store / Open API / Json

도서사이트 '교보문고'의 클론페이지입니다.
페이지 내 도서 콘텐츠는 KaKao에서 제공하는 Open API를 이용했습니다.
상세페이지의 data는 Json을 이용해 처리하여 향후 페이지의 수정,관리의
편의성을 높였습니다.

한 페이지에 많은 정보를 간결하게 나타내도록 하였습니다.
